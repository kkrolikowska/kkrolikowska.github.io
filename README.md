# kkrolikowska.github.io

Personal portfolio site built with React, TypeScript, and Vite.

## Features

- Hero section with introduction
- Work experience timeline
- Skills overview
- Hobbies section
- Responsive design with lazy-loaded sections

## Tech stack

- React 18 + TypeScript
- Vite
- GitHub Pages

## Local development

```bash
npm install
npm run dev
```

## Build & deploy

```bash
npm run build
```

The site is automatically deployed to GitHub Pages on push to `main`.
